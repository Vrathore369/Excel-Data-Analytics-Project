# 🚴 Bike Sales Data Analysis Dashboard (Excel)

## 📌 Project Overview
This project focuses on analyzing bike sales data using Microsoft Excel. The goal is to identify customer purchasing behavior and key factors influencing bike purchases.

An interactive dashboard was created using Excel tools like Pivot Tables, Charts, and Slicers to provide clear business insights.

---

## 🎯 Objectives
- Analyze customer demographics and purchasing patterns
- Identify factors affecting bike purchases
- Create an interactive dashboard for better decision-making

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning & Transformation

---

## 📊 Dashboard Features
- Bike Purchase Analysis (Yes/No)
- Income-wise Purchase Trends
- Age Group Analysis
- Commute Distance Impact
- Region-based Insights
- Interactive Filters using Slicers

---

## 🔍 Key Insights
- Customers with higher incomes are more likely to purchase bikes
- Middle-aged individuals show higher purchase rates
- Short commute distance customers tend to buy bikes more frequently
- Certain regions show a stronger demand for bikes
