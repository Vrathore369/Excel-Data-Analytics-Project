📊 Consumer Complaint Analysis Dashboard (Excel)

## 📌 Project Overview
This project analyzes consumer complaint data using Microsoft Excel to identify patterns, response efficiency, and key issues faced by customers. The goal is to generate insights that help improve customer service and decision-making.

---

## 🎯 Objectives
- Analyze complaints by product and category  
- Evaluate company response performance  
- Measure response time and delays  
- Identify trends over time  
- Highlight areas for improvement  

---

## 🛠️ Tools & Techniques Used
- Microsoft Excel  
- Data Cleaning & Preprocessing  
- Pivot Tables  
- Pivot Charts  
- Slicers (for interactivity)  
- Conditional Formatting  

---

## 📊 Dashboard Features
- 📌 Complaints by Product (Bar Chart)  
- 📌 Company Response Distribution (Pie Chart)  
- 📌 Monthly Complaint Trend (Line Chart)  
- 📌 State-wise Complaint Analysis  
- 📌 Timely vs Delayed Response  
- 📌 Response Time Calculation  

---

## 🔍 Key Insights
- Certain products generate higher complaints  
- Some states contribute more complaints than others  
- Delayed responses reduce customer satisfaction  
- Monthly trends help identify peak complaint periods
