# 📊 Subscription Cohort Analysis Dashboard (Excel)

## 📌 Project Overview

This project focuses on analyzing **subscription-based customer data** using **Microsoft Excel only**.
The dashboard provides insights into **customer behavior, retention patterns, and subscription trends** through Cohort Analysis.

The goal is to transform raw data into **actionable business insights** using Excel’s analytical capabilities.

---

## 🎯 Objective

* Analyze **subscription growth trends**
* Track **paid vs unpaid users**
* Identify **cancellation patterns**
* Understand **user behavior over time**
* Build an **interactive dashboard using Excel**

---

## 🛠️ Tools Used

* Microsoft Excel 2021
* Pivot Tables
* Pivot Charts
* Slicers (Filters)
* Data Cleaning Techniques
* Basic Excel Functions

---

## 📂 Dataset Description

The dataset includes:

* Customer ID
* Subscription Date
* Payment Status (Paid / Unpaid)
* Cancellation Data
* Time-based fields (Month, Year, Day)

---

## 📊 Dashboard Features

### 🔹 KPI Cards

* Total Number of Subscriptions
* Number of Subscriptions Cancelled
* Paid Subscriptions (%)
* Unpaid Subscriptions (%)

### 🔹 Visualizations

* 📈 Monthly Subscription Trend
* 🥧 Paid vs Unpaid Distribution
* 📊 Subscriptions by Day
* 📉 Subscription Cancellations Over Time

### 🔹 Filters

* Year-wise filter using slicer for dynamic analysis

---

## 📈 Key Insights

* Majority of users are **paid subscribers (~96%)**
* A significant number of cancellations observed over time
* Certain days show **higher subscription activity**
* Monthly trends help identify **growth patterns**

---

## 🚀 How to Use

1. Download the Excel file from this repository
2. Open in **Microsoft Excel 2021 or later**
3. Use the **Year slicer** to filter data
4. Interact with charts to explore insights
